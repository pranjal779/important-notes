The image illustrates SNI (Server Name Indication).  
It shows an Application Load Balancer (ALB) using SNI to host multiple websites on a single IP address and port.  
When a client requests a specific hostname (e.g., ), the ALB uses the SNI extension in the TLS handshake to present the matching SSL certificate and route the traffic to the correct target group.
